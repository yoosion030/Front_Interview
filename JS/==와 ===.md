# ==와 ===의 차이가 무엇인가요 🤷‍♀️

## ==
> 값을 비교

```JS
0 == false // true
```
0 값은 false와 동일하므로 true 출력 
``` JS
2 == "2" // true
```
값이 똑같기 때문에 true


``` JS
null == undefined // true
```


## ===
> 값과 자료형을 비교 -> 엄격한 비교
``` JS
0 === false // false
```

0의 자료형 = number  
false의 자료형 = boolean 이기 때문에 false 출력

``` JS
2 === "2" // false
```
자료형이 다르기 때문에 false

``` JS
null === undefined // false
```

## 권장사항
변수를 비교하거다 어떤 비교를 위해 항상 '===' 연산자를 사용할 것을 권장한다.
> '=='로 자동으로 형변환 시키는 것 보단 직접 자료형을 변환시켜 코드의 가독성을 높이기 위해
